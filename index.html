À propos

Le petit livre sur Redis est un livre gratuit d'introduction à Redis.

Le livre a été écrit par Karl Seguin, avec l'assistance de Perry Neal.

Cette traduction française a été réalisée par David Loureiro, Augustin Ragon et Haikel Guémar

Si vous aimez ce livre, peut-être aimerez-vous Le petit livre sur MongoDB, traduction française réalisée par les mêmes personnes de The Little MongoDB Book, la version originale, aussi écrite par Karl Seguin, avec l'assistance de Perry Neal.

Formats

Ce livre est rédigé en Markdown et converti en PDF en utilisant Pandoc.

Le template TeX fait usage de Lena Herrmann's JavaScript highlighter.

Les formats Kindle et ePub sont fournis par Pandoc.

Génération des livres

Les paquets listés ci-dessous sont pour Ubunti. Si vous utilisez un autre Système d'Exploitation, les noms devraient être similaires.

PDF

Dependances

Paquets:

  • pandoc
  • texlive-xetex
  • texlive-latex-extra
  • texlive-latex-recommended

Vous devriez aussi avec les polices Microsfts installées. Vous pouvez cependant changer les polices dans le fichier common/pdf-template.tex si vous le souhaitez.

Construction

Exécutez make fr/redis.pdf.

ePub

Dependances

Paquets:

  • pandoc

Construction

Exécutez make fr/redis.epub.

Mobi

Dependances

Paquets:

  • pandoc

Vous devez aussi avoir KindleGen d'installé.

Construction

Lancez make fr/redis.mobi.
